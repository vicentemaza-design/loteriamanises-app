## Notas de Autorización y Uso
*   **DGOJ:** El uso de estos logotipos está sujeto a las directrices de imagen institucional de la Dirección General de Ordenación del Juego.
*   **Validación Legal:** Se requiere validación final por parte del departamento legal del cliente para asegurar que los tamaños, colores y ubicaciones cumplen con el Real Decreto de comunicaciones comerciales de las actividades de juego.
*   **Sustitución:** Estos assets deben usarse como referencia. Si se requiere una versión vectorial (SVG), se recomienda solicitarla directamente a la DGOJ o convertirlos manteniendo la fidelidad cromática.
